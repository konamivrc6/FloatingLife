# 5 落水

然后刘宁梧连同着手中拿着的键盘消失了。拖鞋还在原位，手机还放在柜台上。

刘宁梧先感到一阵失重，大概两三秒。

水中远处传来了一些尖锐的噪声（暗示水体特殊声学效应，与平时水中沉闷的声音不同）。这时，感到有什么东西抓住了她。吓得睁开眼，发现是人，给她带上了一个透明的面罩，闪着蓝光。刘宁梧感到空气充满了口鼻，能呼吸了，眼睛也逐渐适应了水下环境。发现自己悬浮在水中，周围围着几个人，用听不懂的语言在向她说话，用手机翻译成各种语言在试，看刘宁梧能听懂哪种语言。

# 6 空气流

夏满：“刘宁梧怎么还没有回来？”去杂物间看，尽可能轻地推门，站在门口望，发现里边没有人，也没有第二个出口。夏满感到不妙。

夏满紧张。夏满去找陈观：“看到刘宁梧了吗？”

陈观：“没有呀。我看看店里的视频监控。”

店里没有杂物间的监控。只能看到刘宁梧进了杂物间之后就没出来过。

夏满从货架上拿了一本花之书和一页“空气分析”，把花之书打开到封底，把空气分析放到书页的位置，然后空气分析就和书脊自动连上了。夏满来到杂物间门口，把花之书在地上展开，激活“空气分析”，设置半径 6 m，扫描周围的空气。夏满感觉自己的许多魔力被花之书抽走了，瞬间变累。

告诉读者空气能量记忆的特性，但不要展开，用“空气有记忆，记性不好”这一句说就够了，避免打断节奏。

两三分钟后，扫描完成。夏满启动“空气分析”的分析功能。进度条显示剩余 3.6 h。

夏满又从货架上取了一个“叶吟处理器算力页”，插到花之书里，重启花之书，显示检测到叶魔法的叶吟处理器。然后重新启动分析。一分钟后，分析结果出来了，三维全息显示在魔法书摊开的页面上方。从分析结果看来，刘宁梧蹲下碰了拖鞋，然后拖鞋放出传送法阵，传送刘宁梧去了东北方（传送用到了巨量魔力，在空气里留下了极深的印记，延伸向东北方）。

陈观也看了结果，立刻报警。（不要用“电话”这个词，“用手机联系警察”，视频通话）

警察不信，而且说失踪十二小时才立案。

夏满想：

谁设下的机关？店内员工最可疑，但肯定问不出来，现在问也没意义。这种东西要么是陷阱，要么是密道。如果是陷阱，刘宁梧有危险。如果是密道，那拖鞋就是“钥匙”，但也有疑点，刘宁梧为什么会知道拖鞋是机关？有人提前和刘宁梧约好？不，刘宁梧不会什么都不给夏满说。刘宁梧一定是误触发密道机关。如果密道对面有人，看到来的人是刘宁梧，秘密即泄露，会灭口？

不管怎么说，刘宁梧有危险。警察若是十二小时之后才来，刘宁梧可能……

夏满立刻取了手机，她的和刘宁梧的。从货架上拿两颗装满魔力的蓝宝石，一瓶保命药水，手台。

陈观还在和警察沟通，尽力说服警察相信他。陈观看见夏满：“你干什么！别冲动！”但没拦住。夏满跑到杂物间，拾起花之书，蹲下，深吸气，闭眼，摸拖鞋。

但什么都没发生。夏满攥紧拖鞋。

陈观从后边过来：“夏满！”夏满转身站起，对着陈观的手机，向警察说：“相信我们，这是真的！”

警察：“几天前有个小孩报假警也这么说的。哪有人这么随意地用传送，还是自设的术式，贵死了。”

这时，夏满的手机响起。

# 7 “我还会再回来的……”

切换至澈轮场景。

手上缠着编织线的那个人：“阿比盖尔刚刚告诉我，从口音上判断，你说汉语。你是否需要医疗服务？”

刘宁梧：“啊，谢谢，不用了。我没有受伤。”向四下看看，“这个面罩，我需要付钱吗？”

周围的其中一个人，想必是给她戴面罩的人，用手机翻译：“不用。面罩是你依法享有的。它至少能用三天。愿万水澈轮护佑你。”

见到刘宁梧无大碍，附近聚集起来的人渐渐散去。刘宁梧想：夏满，得尽快告诉她，尽快回去。

Weave 提示刘宁梧：「阿比盖尔想要通过 WeaveOS 与您交谈」

刘宁梧同意。阿比盖尔的影像出现在视野中，平涂插画风格（二次元），描写。阿比盖尔的影像虽然是平涂插画风格，但还是很真实，能和环境互动（当然无法造成实际的影响），衣服和头发还能被水流推动。

“初次见面。虽然不知道您是怎么来到这里的，但您放心吧，澈轮是全世界最安全的地方。在水下一定要注意呼吸设备。我是阿比盖尔，世界最强的人工智能，由水原科技研发，治理澈轮。您初访本国，需要提供您的基本信息。我想用您的编织线序列号从水原科技处获得您的相关信息，再联系您原先所在地的政府进行验证，可以吗？”

刘宁梧：“麻烦你了。”

阿比盖尔：“谢谢您的配合，刘宁梧。我没认错吧？”

刘宁梧：“这么快，真厉害。”

阿比盖尔：“谢谢。现在我能帮您什么呢？去附近买一套水下的衣服……”

刘宁梧打断：“先让我联系一下朋友。”

然后刘宁梧用 Weave 联系夏满。

夏满在手机上看到繁荣的水下都市，但看不到刘宁梧。

刘宁梧：“夏满，我在澈轮，很安全。”

写出刚联系上时二人的欣喜与激动。

但夏满很快冷静。视频通话可能被伪造。看看设备验证，确实是刘宁梧的 Weave 发出的视频通话。Weave 没有摄像头，所以夏满只能看到刘宁梧的视野，看不到刘宁梧其人，很合理。Weave 也不可能被其他人使用，所以真的是刘宁梧。

阿比盖尔通过这次通话了解到了信息。阿比盖尔提出免费用洋流送刘宁梧回到煎州。刘宁梧欣然同意。

刘宁梧捡起来浮在水中的之前掉了的鞋和键盘。

描写使用洋流。用带空气泡的。

坐上洋流。有路人用魔法把刘宁梧的衣服弄干。

飞出澈轮。刘宁梧在空中巨大的水流中。

刘宁梧从空中看澈轮，当然看不到全貌。像巨大的蓝宝石一样浮在空中。写出震撼感。

告诉读者洋流网不仅覆盖了整个澈轮，还扩散到澈轮以外的广阔区域，为半个世界提供服务。

在洋流中，刘宁梧和夏满整理已知信息。但由于刘宁梧移动速度超快，网络状况挺不好的，时不时断联。夏满无意识地把那个拖鞋放到了柜台后边的地上。

转换视角。在瓦尔基里灌注站，午高峰，店内人多且杂。夏满心系刘宁梧，在前台站着发呆。顾客在夏满面前叫了三声，夏满才回过神来。

这时，一个看起来很普通的人去了趟杂物间，然后又是微风把杂物间的门吹动了几下。这个人十几分钟后才出来。整个过程中没人注意到他。他的举止中带着十分稳重的气质，要从字里行间写出。

刘宁梧的洋流切换到了支流，不久就到了煎州的洋流站。在洋流站，气泡包裹着刘宁梧不断减速，最后从一面水墙中“胞吐”出来，脚终于踩在了地上。刘宁梧扫视一圈，看到

```
绪国公民  入境
会馆 - 绪国海关
```

的通道标识，正准备往那里走，接受边防检查。几个警察直接迎了上来。

带队的是李警官，女，干练。向刘宁梧自我介绍。李警官叫作李旷诚，这个名字只在自我介绍时出现一次，之后文本中都用“李警官”。

李警官让刘宁梧安心，要求刘宁梧跟她来。刘宁梧跟着李警官走，前后都有警察护着，免了复杂的入关流程，直接到了出口处。李警官让刘宁梧上警用飞艇，说是要护送刘宁梧，问刘宁梧回家还是去什么别的地方。刘宁梧要去瓦尔基里灌注站。

在路上，李警官说，阿比盖尔联系到了煎州警察。没有走外交渠道，就是普通地发了手机通知，而且直接通知到南湖区派出所，没给上级说。警察这才知道，方才陈观和夏满所说均为真。

（澈轮很强大，绪国不敢惹，不能造成“外交事故”。南湖区派出所的警察们也不想让事情闹大，而且看阿比盖尔根本没给上级说，那意思也应该是要大事化小，小事化了的）

李警官和刘宁梧一行人来到瓦尔基里灌注站，看到几个警员正在用十分高级的设备扫描整个灌注站的空气。设备主体是一个黑漆的长方形金属盒子，接了两块大蓝宝石。所有无关人员，包括陈观、夏满、小王、小焦，都被请出了灌注站，因为扫描的时候人不能进店，干扰空气记忆。扫描了一个多小时。扫描期间，因为外边很热，李警官让灌注站的人都坐在飞艇上吹空调。

片刻后，李警官也上了飞艇，手里提着五杯奶茶，给灌注站的五个人。

扫描完后，警察来找陈观调监控，发现没有刘宁梧传送走之后的监控。陈观：“可能是我当时查监控的时候不小心点了暂停吧。”

挺合理的。缺失的视频内容也是在刘宁梧传送走之后的，对案件基本没什么影响。警察们走时，送了瓦尔基里灌注站一个上好的键盘来道歉。

之后，刘宁梧发现那个拖鞋还放在前台后边的地上。夏满：“警察们怎么能漏了这么重要的东西！”

# 9 官方说辞

夏满到了派出所。说明来意后，见到了李警官。

### 拖鞋相关

拖鞋是花按在多年前放置的，当时还没有 Weave，目的是在应急情况下及时逃跑，所以目的地选的是澈轮。检测花按的方法是，检测花按脑中的意识集群术式结构——非常好的一个检测位点，准确度极高，卡文迪许实验室的花按补丁也是通过这种方法检测花按的。不过，由于 Weave 在结构上与意识集群有类似之处，所以拖鞋也会把装有 Weave 的人当成花按给传送了。至于为什么传送系统没有误杀使用 Weave 的人，自然是因为卡文迪许实验室在 Weave 发布前做了测试，调整了检测位点，检测更具有特异性的意识集群术式结构。

拖鞋在启动后，会发出消息。所有花按都能收到。所以花按第一时间就做了反应。刘宁梧被传送走那天，瓦尔基里灌注站中午来的那个人是花按，但不是四方天坛的那一个。他来掩盖痕迹。怎么掩盖呢？空气记忆有最大的记忆数量。向传送法阵这种大型法术，一次就可以用掉许多记忆容量。花按先是通过电脑后门远程停掉了监控，把陈观查监控之后的所有视频都删除，伪造陈观不小心暂停监控的事件。然后趁着中午食客很多，易容进入瓦尔基里灌注站杂物间。然后模仿刘宁梧，用一次寻物术式，然后拿起一个东西，用类似魔术的手法把一卷 Vector 2 放到货架上。花按按照计算好的量，对齐时间，精确地同时触发身上带着的几十个 Vector 2（这烧掉了价值数亿元的魔力），没有载荷，目的地指向东北方的一个无人区。这样就把空气记忆洗到刚好刘宁梧传送走之后。这样就几乎完美地伪造了痕迹。

为什么选 Vector 2？因为花按知道 Vector 3 可能会杀死他。（花按之前听说有其他人被用卡文迪许实验室用传送阵秘密处死了。）而且 Vector 2 比较旧，出点问题更合理。

Vector 2 的热量有 99.9 % 都释放在目的地，所以杂物间没有变成熔炉。

由于这些所有的事情都在几个小时内发生，空气记忆都很清晰，很难通过噪声的大小来判断时间先后，只能通过空气的位移反推事件发生的次序先后，又因为杂物间环境密闭，空气不咋动，于是也就避开了时间漏洞。密闭的环境，一方面使空气记忆更易保存，又方便了花按覆盖，这样就骗过了所有人。
